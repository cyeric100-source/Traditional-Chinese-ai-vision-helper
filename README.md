# 由 AI 撰寫的「AI 智慧助理」使用手冊

## 系統需求與安裝

*   **系統需求**：任何安裝有現代網頁瀏覽器（如 Chrome, Edge, Safari, Firefox）的電腦或行動裝置。
*   **安裝方式**：本程式為單一 HTML 檔案。無需安裝，直接開啟檔案即可使用。
*   **網路連線**：必須連接網際網路才能呼叫 AI 服務。

---

## 介面佈局概覽

本程式由上而下分為三個主要區域，均設有標題層級以便導航：

1.  **標題區**：顯示程式名稱。
2.  **設定區域 (標題 1)**：包含 AI 服務商選擇、API 金鑰輸入、模型清單與系統指令。首次使用時會自動展開，之後預設為收合狀態。
3.  **對話結果 (標題 1)**：AI 的回答會顯示於此。新訊息會依序向下排列。
4.  **底部控制列 (地標：操作區)**：包含功能開關、檔案上傳按鈕、文字輸入框與傳送按鈕。此區域固定於視窗底部。

## 快速鍵列表

為了提升操作效率，程式內建以下 AccessKey 快速鍵（Windows 通常搭配 `Alt`，Mac 通常搭配 `Control` + `Option`）：

*   **Alt + S**：跳轉至「設定」區域並展開/收合。
*   **Alt + U**：觸發「拍攝/檔案」上傳功能。
*   **Alt + I**：跳轉至「文字輸入框」。
*   **Alt + C**：複製最後一次 AI 的回答結果（僅在有結果時有效）。

## 開始使用

### 1. 圖片或檔案辨識
1.  按下 **Alt + U** 或點擊「📷 拍攝/檔案」按鈕。
2.  選擇電腦中的圖片（JPG, PNG）或 PDF 文件。
3.  **自動模式（預設）**：選擇檔案後，系統會自動將圖片傳送給 AI 進行分析。
4.  **暫停模式**：若勾選「暫停傳送」，選擇檔案後需手動按下「傳送」按鈕才會開始分析。

### 2. 文字對話
1.  按下 **Alt + I** 進入輸入框。
2.  輸入您的問題。
3.  按下 Enter 鍵或「傳送」按鈕。

### 3. 獲取結果
*   當 AI 處理完成後，螢幕閱讀器焦點會自動跳轉至「對話結果」標題，並提示有新內容。
*   若需將結果分享或貼上至其他程式，可按下 **Alt + C** 複製內容。

## 設定與模型選擇

按下 **Alt + S** 可展開設定區域。

### 1. 選擇 AI 服務商
程式支援多種 AI 來源，建議依需求選擇：
*   **Google Gemini**：推薦使用。速度快、免費額度高，支援圖片與 PDF 分析。
*   **OpenRouter**：整合型服務。可連接各大模型（如 Claude, Llama），相容性佳。
*   **Groq / OpenAI / Mistral**：官方 API。
    *   *注意*：除 Gemini 與 OpenRouter 外，部分官方 API 因瀏覽器安全性限制（CORS），可能無法直接更新模型清單，需手動輸入模型 ID。

### 2. API Key (金鑰)
在對應欄位輸入您的 API Key。系統會自動儲存於瀏覽器中，下次開啟無須重新輸入。

### 3. 更新模型清單
*   在支援的服務商（如 Gemini, OpenRouter）模式下，點擊「🔄 更新清單」按鈕，可從網路獲取最新模型。
*   選擇完成後，點擊「💾 儲存」按鈕確認。

## 如何獲取 API Key

要使用本工具，您需要向 AI 服務商申請金鑰（API Key）。以下是各家服務商的申請方式：

### 1. Google Gemini (Google AI Studio)
*   **特點**：目前提供免費額度，速度快，申請最容易。
*   **申請網址**：[https://aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)
*   **步驟**：登入 Google 帳號後，點擊「Create API Key」按鈕即可取得。

### 2. OpenRouter
*   **特點**：整合了數十種模型（含免費與付費），只需一組 Key 即可使用多種 AI。
*   **申請網址**：[https://openrouter.ai/keys](https://openrouter.ai/keys)
*   **步驟**：註冊帳號後，點擊「Create Key」。

### 3. Groq
*   **特點**：回應速度極快，目前提供相當大方的免費測試額度。
*   **申請網址**：[https://console.groq.com/keys](https://console.groq.com/keys)
*   **步驟**：登入後點擊「Create API Key」。

### 4. OpenAI (ChatGPT)
*   **特點**：模型聰明（GPT-4o），但通常需要綁定信用卡並付費才能使用 API。
*   **申請網址**：[https://platform.openai.com/api-keys](https://platform.openai.com/api-keys)

### 5. Mistral AI
*   **特點**：歐洲頂尖 AI 公司，提供免費試用額度。
*   **申請網址**：[https://console.mistral.ai/api-keys](https://console.mistral.ai/api-keys)

### 6. Perplexity
*   **特點**：專精於聯網搜尋，需綁定信用卡付費。
*   **申請網址**：[https://www.perplexity.ai/settings/api](https://www.perplexity.ai/settings/api)

## 常見問題與故障排除

### Q1: 按下「更新清單」後出現錯誤？
*   請檢查 API Key 是否正確。
*   部分服務商（如 OpenAI 官方）不支援從網頁直接抓取清單，這是正常的安全限制。請直接在「模型 ID」欄位輸入已知模型名稱（如 `gpt-4o`）。

### Q2: PDF 無法上傳？
*   目前僅 **Google Gemini** 模式支援直接上傳 PDF 文件進行分析。其他模式請先將文件轉為圖片或純文字。

### Q3: 聲音提示「設定已儲存」或「已複製」？
*   程式會在執行儲存、複製、上傳等動作時，透過螢幕閱讀器發出簡短的狀態提示，這些提示大約會顯示 1.5 秒後自動消失，以免干擾閱讀。

### Q4: 什麼是「聯網」功能？
*   勾選底部控制列的「聯網」選項後，AI（如 Gemini）會嘗試搜尋網路上的最新資訊來回答問題。若不需要即時資訊，可取消勾選以加快回應速度。
